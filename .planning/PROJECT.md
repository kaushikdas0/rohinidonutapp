# Rohini Donut App

## What This Is

A fun, single-page web app featuring a CSS/SVG animated 3D spinning donut with a dark + neon cyber aesthetic. Built as a creative coding experiment with a polished style guide foundation using React, Vite, and Material UI. Design-first approach — style guide defined before implementation, with Stitch used for design ideation.

## Core Value

A visually striking animated donut experience with a cohesive, well-defined design system underneath it.

## Context

- Fun experiment / creative coding project
- Design-first workflow: style guide → Stitch ideation → implementation
- The donut animation is CSS/SVG based (pseudo-3D with transforms), not WebGL
- Phosphor Icons chosen for their flexible weight system and playful feel that matches the vibe
- Dark + neon palette inspired by cyberpunk/retro-futuristic aesthetics

## Constraints

- **UI Framework**: Material UI (MUI) — chosen for theming system and component library
- **Build Tool**: Vite — fast dev experience for React
- **Icons**: Phosphor Icons — open source, flexible weights
- **Font Style**: Playful/rounded (Nunito, Comfortaa, or Quicksand family)
- **Animation**: CSS/SVG only — no WebGL or canvas dependencies
